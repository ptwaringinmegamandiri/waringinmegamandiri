# PT Waringin Mega Mandiri – Website Perusahaan Konstruksi

## 1. Project Description
Website profesional PT Waringin Mega Mandiri, perusahaan konstruksi yang menonjolkan identitas modern, futuristik, dan berbasis teknologi tinggi (BIM, Digital Construction, Smart Building). Target pengguna: calon klien korporat, investor, mitra bisnis, dan pelamar kerja.

## 2. Page Structure
- `/` - Homepage (Hero, Layanan, Proyek Unggulan, Stats)
- `/tentang-kami` - Tentang Kami (Profil, Visi Misi, Nilai, Tim)
- `/portofolio` - Portofolio Proyek (Grid, Filter, Detail)

## 3. Core Features
- [x] Homepage dengan hero futuristik & animasi
- [x] Navigasi responsif dengan efek scroll
- [x] Section layanan dengan icon modern
- [x] Carousel proyek unggulan
- [x] Halaman Tentang Kami dengan timeline & tim
- [x] Portofolio dengan filter kategori
- [x] WhatsApp floating button
- [x] Footer lengkap dengan navigasi
- [ ] Halaman Layanan (detail)
- [ ] Halaman Sertifikat & Legalitas
- [ ] Halaman Karir (dengan form apply)
- [ ] Halaman Blog/News
- [ ] Halaman Kontak (form + peta)

## 4. Data Model Design
Static/mock data only (no backend needed for Phase 1)

## 5. Backend / Third-party Integration Plan
- Supabase: Tidak diperlukan di fase ini
- Shopify: Tidak diperlukan
- Stripe: Tidak diperlukan
- Form Karir & Kontak: Akan menggunakan Readdy Form URL

## 6. Development Phase Plan

### Phase 1: Core Pages (CURRENT)
- Goal: Membangun 3 halaman utama (Homepage, Tentang Kami, Portofolio)
- Deliverable: Website dapat dilihat dan navigasi antar halaman

### Phase 2: Halaman Pendukung
- Goal: Layanan, Sertifikat, Blog, Karir, Kontak
- Deliverable: Website lengkap dengan semua halaman

### Phase 3: Integrasi & Optimasi
- Goal: Form kontak/karir aktif, SEO, optimasi performa
- Deliverable: Website siap publish
